# Clipéo

Clipéo est une application Windows open source permettant d’analyser une URL et de télécharger les formats proposés par la source : vidéo, MP3, M4A, WAV ou miniature.

## Principes

- Traitement entièrement local avec yt-dlp, Deno et FFmpeg.
- Aucune URL ni donnée personnelle envoyée à Bob59.
- Utilisation réservée aux contenus personnels, libres, autorisés ou officiellement téléchargeables.
- Aucun contournement de DRM ou de mesure de protection.

## Développement

```bash
npm install
npm run start
npm run dist
```
